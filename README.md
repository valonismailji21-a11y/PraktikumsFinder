# PraktikumsFinder LU/ZG Cloud-PWA

Diese Version ist eine statische iPhone-PWA. Sie benötigt nach dem Hosting keinen lokalen Windows-PC-Server mehr.

## Schnellstart

1. Dateien in ein GitHub-Repository hochladen.
2. GitHub Pages auf Branch `main` und Ordner `/ (root)` aktivieren.
3. Optional GitHub Actions aktivieren, damit `data/jobs.json` automatisch aktualisiert wird.
4. GitHub-Pages-Link auf dem iPhone in Safari öffnen.
5. Über Teilen → Zum Home-Bildschirm installieren.

Die vollständige Anleitung steht in:

`ANLEITUNG_IPHONE_OHNE_PC_SERVER.md`
